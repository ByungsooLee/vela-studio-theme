# Image Library

このフォルダは、指示ベースで差し込みたい画像の保管場所です。

運用ルール:

- 新しい画像は `image-library/` に格納
- 配置先の指示は、画像ファイル名と対象（例: `トップHero`, `商品カード1枚目`）をセットで指定
- Shopifyで実際に表示に使う場合は、必要に応じて `assets/` 側へ反映してテーマコードに接続

例:

- `image-library/hero-spring-01.jpg` をトップHeroに設定
- `image-library/pdp-alt-02.webp` をPDPサムネイル2枚目に使用
