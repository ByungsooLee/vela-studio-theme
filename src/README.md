# src/ — デザインプロトタイプ（非稼働）

このディレクトリはReact/TypeScriptで書かれたUIコンポーネントのプロトタイプです。
**Shopifyテーマには組み込まれていません。**

## 用途
- デザイン検討用のリファレンス実装
- `src/styles/tokens.css` のCSS変数は `assets/pawlux-tokens.css` の元になっている

## AI作業時の注意
このディレクトリのファイルは読まなくてよい。
